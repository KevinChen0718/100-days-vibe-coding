# 100 天 Vibe Coding 馬拉松

連續 100 天，每天做一個有趣、可以對外公開的網頁小專案，累積自己的 Vibe Coding 作品集。

每個專案都是一個獨立資料夾，盡量做到**打開瀏覽器就能玩／用**，零安裝、零依賴。

- 每個 `day-XXX-*/` 資料夾都是一個獨立、可直接打開 `index.html` 的作品。
